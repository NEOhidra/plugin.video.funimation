plugin.video.funimation
=======================

An XBMC plugin for streaming videos from funimation.com

Install
-------
clone the repo into your xbmc/addons folder

    cd ~/.xbmc/addons
    git clone https://github.com/Sinap/plugin.video.funimation.git
